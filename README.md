# Tabletop.site

A versatile tabletop RPG application designed to assist game masters and players in game preparation and live sessions.

Универсальное приложение для настольных ролевых игр, созданное для помощи мастерам и игрокам в подготовке и проведении игровых сессий.

## Features / Возможности

- Support for multiple tabletop RPG systems / Поддержка множества систем настольных ролевых игр
- Tools for game masters and players / Инструменты для мастеров игры и игроков
- Session management and preparation utilities / Управление сессиями и утилиты для подготовки

## Supported Languages / Поддерживаемые языки

- English / Английский
- Russian / Русский
